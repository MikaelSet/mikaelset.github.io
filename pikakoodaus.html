Koodikieliä

Html

HTML eli hypertext markup language on esim. internet-sivujen kirjoittamisessa käytetty markup kieli. Suurin osa nettisivuista on koodattu HTML kielellä.

CSS

CSS eli Cascading Style Sheet on esim. verkkosivujen tyylin muokkaamiseen tarkoitettu kieli. Sitä käytetään yleensä HTML:än kanssa jotta nettisivut näyttäisivät hyvältä.

JavaScript

JavaScript on pääasiassa verkkoympäristössä käytettävä ohjelmointikieli. JavaCript:illä voi lisätä verkkosivuille dynaamista toiminnallisuutta.

Python

python on yksi maailman suosituimmista ohjelmointikielistä jota käytetään esim. tietokoneohjelmien ja pelien koodaamisessa.

C++

C++ on C-ohjelmointikieleen perustuva kieli. C++:lla on kirjoitettu suuri osa maailman ohjelmistoista, mukaan lukien käyttöjärjestelmät, lääketieteelliset ohjelmistot, simulointiohjelmistot, finanssialan ohjelmistot, tietokoneavusteisen suunnittelun ohjelmistot, sulautetut järjestelmät, tietokoneverkot, ja videopeliohjelmistot (mm. pelimoottorit).
