JavaScript

JavaScript on dynaaminen ohjelmointikieli, jota käytetään usein verkkosivujen ja sovellusten kehittämisessä.

Python

Python on yleiskäyttöinen ohjelmointikieli, joka on tunnettu selkeydestään ja luettavuudestaan.

Java

Java on tehokas ja monipuolinen ohjelmointikieli, jota käytetään laajasti erilaisissa sovelluksissa.

C#

C# on Microsoftin kehittämä ohjelmointikieli, joka on erityisesti suunniteltu Windows-sovellusten kehittämiseen.

Ruby

Ruby on dynaaminen ja joustava ohjelmointikieli, jota käytetään muun muassa web-sovellusten kehittämisessä.

Tämä on esimerkki nettisivusta, joka esittelee viittä eri koodauskieltä.
